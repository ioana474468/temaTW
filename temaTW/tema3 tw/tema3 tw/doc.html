Introducere

Acest document oferă o descriere detaliată a codului JavaScript care utilizează Dog API pentru a obține imagini aleatoare cu câini și efectuează diverse operațiuni de manipulare a acestora, cum ar fi oglindirea, ajustarea luminozității etc. Codul este structurat într-un mod modular și curat pentru a îmbunătăți citibilitatea și întreținerea.

Funcția "getRandomDog"

Această funcție asincronă utilizează Dog API pentru a obține o imagine de câine la întâmplare. Aceasta afișează imaginea într-un container HTML, într-un canvas și oferă link-ul imaginii.

  • fetch('https://dog.ceo/api/breeds/image/random'): Realizează o cerere către Dog API pentru a obține o imagine aleatoare de câine.
  • "response.json()": Transformă răspunsul în format JSON.
  • "displayDogImage(data.message)": Afisează imaginea într-un element HTML.
  • "displayDogImageInCanvas(data.message)": Afisează imaginea într-un canvas.
  • "document.getElementById('link').innerHTML=data.message;": Afișează link-ul imaginii.

Funcțiile de Afisare a Imaginilor

  1. "displayDogImage(imageUrl)"
    • Obține elementul HTML în care va fi afișată imaginea.
    • Creează un element imagine și setează sursa imaginii.
    • Afișează imaginea în container.
  2. "displayDogImageInCanvas(imageUrl)"
    • Creează o imagine, setează atributul crossOrigin pentru manipularea imaginii, și așteaptă încărcarea acesteia.
    • După încărcare, ajustează dimensiunile canvas-ului și desenează imaginea în canvas.

Event Listener pentru Oglindirea Imaginii

  • Listener pentru butonul "Optiunea 1" și "Optiunea 2".
  • Utilizează funcția "mirrorImage(imageData, k)" pentru a oglindi imaginea în canvas în patru etape, cu întârzieri de 1 secundă.

Funcția de Oglindire a Imaginii (mirrorImage)

  • "mirrorImage(imageData, k)": Oglidează imaginea pe orizontală în patru etape, luând în considerare regiunile corespunzătoare ale canvas-ului.

Event Listener pentru Ajustarea Luminozității

  • Listener pentru butoanele "Mărește Luminozitatea" și "Scade Luminozitatea".
  • Utilizează funcția "adjustBrightness(brightnessChange, k)" pentru a ajusta luminozitatea imaginii în patru etape, cu întârzieri de 1 secundă.

Funcția de Ajustare a Luminozității (adjustBrightness)

  • "adjustBrightness(brightnessChange, k)": Modifică luminozitatea imaginii în patru etape, luând în considerare regiunile corespunzătoare ale canvas-ului.

Concluzie

Acest cod JavaScript demonstrează o integrare eficientă a Dog API pentru a obține imagini de câini și efectuarea de manipulări asupra acestora într-un mod interactiv. Structura modulară și comentariile explicative facilitează înțelegerea și extinderea codului în viitor.
